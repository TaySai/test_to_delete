3 ways to get data

  • Just by navigating in the website
  • By arguments in url, return JSON "example /api/v1/web?messier=example1
    " \ "example /api/v1/web?galaxy=example2
    " \ "example /api/v1/web?messier=example1&galaxy=example2"
  • By batch with json application, return JSON can be parsed as list 
    import requests as rq
data = [{'some': 'data'}, {'messier_name':'m31'}, {'galaxy_name':'and'}]
r = rq.get('http://127.0.0.1:5000/api/v1/batch/', json=data, headers={
    'Content-Type':'application/json',
})
